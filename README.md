# Decision Assistant AI 🧠⚡

A Streamlit-based Python web application that helps users make better decisions using Large Language Models (LLMs).

The application accepts a decision problem, evaluates options, analyzes pros and cons, identifies risks, and provides structured recommendations.

---

## 🚀 Features

- Interactive web UI built with Streamlit
- AI-powered decision analysis using LLM
- Pros / Cons comparison
- Risk assessment
- Recommendation scoring
- Scenario-based suggestions
- Easy deployment

---

## 🧩 Use Cases

Examples:

- Should I buy a new laptop?
- Which investment option is better?
- Should I switch jobs?
- Which technology stack should I choose?

---

## 🛠 Tech Stack

- Python 3.10+
- Streamlit
- OpenAI API / LLM Provider
- Pandas
- dotenv
